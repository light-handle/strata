Strata

Visualize every Claude Code session as a 3D topographic terrain. Explore peaks of token usage, browse session history, and resume past conversations — all from your browser, running entirely on your machine.

View on GitHub Get Started
Quick Start
# clone & run
git clone https://github.com/light-handle/strata.git
cd strata
npm install
npm run dev

# open http://localhost:5173

Features

▲

3D Terrain Map

Sessions rendered as Gaussian-smoothed peaks on a heightfield. Contour lines, tactical grid, and a thermal color ramp from navy through teal and amber to gold.

☰

Session Browser

Search and filter all sessions across every project. See first prompts, token counts, models, and duration at a glance. Click to fly to the terrain peak.

⚡

Live Updates

WebSocket connection watches your Claude files in real-time. Start a session in another terminal and watch the terrain grow as tokens flow.

☰

Activity Timeline

D3-powered stacked area chart showing token usage over time, broken down by project. Hover crosshair reveals daily details.

▶

Resume Sessions

One click to resume any past session. Opens a terminal with claude --resume so you can pick up exactly where you left off.

⌘K

Command Palette

Fuzzy search across sessions, projects, and actions. Navigate the entire dashboard from your keyboard.

How It Works

1

Scans your local Claude data

Reads all .jsonl session files from ~/.claude/projects/ — the directory Claude Code uses to store conversation history.

2

Extracts session metadata

Parses every message to extract token usage, tool calls, models, timestamps, prompts, and response summaries.

3

Builds the terrain

Maps sessions onto a 2D grid (time × project) and applies Gaussian smoothing to generate organic terrain peaks from token volumes.

4

Renders in your browser

Three.js renders the heightfield with contour lines and vertex colors. React powers the panels. D3 draws the charts. All connected via WebSocket for live updates.
